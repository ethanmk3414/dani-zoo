# Dani Zoo

A free iPhone-friendly daily love app for Danielle Porter (Dani), from Ethan.

## Run on iPhone
1. Upload these files to a free GitHub repository.
2. Go to Settings > Pages.
3. Source: Deploy from branch, Branch: main, Folder: root.
4. Open the GitHub Pages link on iPhone Safari.
5. Tap Share > Add to Home Screen.

## Run locally
Open `index.html` in a browser.

## Features
- Daily animal
- Daily bow
- Daily love note
- Dani/Ethan personalization
- 30 animals, 20 bows, 25 accessories, 15 colors, 15 backgrounds
- Thousands of combinations
- Favorites
- Streak/unlock counters
- Add custom compliments
- Works offline after opening because it uses local browser storage
